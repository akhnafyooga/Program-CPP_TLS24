CARA MENJALANKAN CODE C++ 
A. MENGGUNAKAN VISUAL STUDIO CODE
   1. OPEN persegi.c++ dengan Visual Studio Code
   2. Klik tombol "run" (logo play) di kanan atas.
   3. Masukkan Panjang dan lebar persegi yang ingin kamu hitung
   4. Klik enter
   5. Code excecution successful

B. MENGGUNAKAN WEB PROGRAMIZ C++ COMPILER
   1. OPEN persegi.c++ jika mempunyai VS Code. Jika tidak, open persegi.txt
   2. COPY semua code dengan select -- ctrl+c
   3. Buka Website Programiz C++ Compiler (link: https://www.programiz.com/cpp-programming/online-compiler/ )
   4. KLIK "RUN"
   5. Masukkan Panjang dan lebar persegi yang ingin kamu hitung
   6. Klik enter
   7. Code excecution successful
